{# ─── PATH 1 · THE GAUNTLET ─── #}
⚔ path 1 · running an existing test

If the CLI wizards don't fit your workflow, you can do it by hand. Run ./agent-arena-cli.py validate after any manual edit.

  1. 01
    pick a test
    Browse /tests/<test name>/ and open its test.yaml to see the description and each stage's prompt.
  2. 02
    run each stage in order
    For each stage, feed the prompt to the LLM exactly as written. Each stage continues from the previous stage's output (the codebase is preserved), but you must start from a fresh agent session every time — no in-memory context, conversation history, or cached planning carries over.
    note · timing Record API time only — the time the agent actually spent calling the model. Do not include time you spent reading responses, approving tool confirmations, sipping coffee, etc. Claude Code: /usage shows exactly this. Other agents: use the equivalent if available; otherwise estimate and say so in the notes.
    note · extras If you used additional MCP servers, skills, custom subagents, hooks, or similar beyond the agent's defaults, document them. Add them to settings: (e.g. mcps: linear,sentry / skills: simplify,review) or describe them in the stage notes.
  3. 03
    create a run directory
    Place it at /tests/<test name>/results/<run-id>/. The run id is a short, unique slug — typically <your-github-username>-<agent>-<model>-<settings>, e.g. tin-cat-claude-code-sonnet-4.6-high-effort. For multiple runs of the same config, append -2 or a date.
  4. 04
    drop in the stage directories
    One subdirectory per stage you ran, named exactly like the stage id in test.yaml (e.g. stage-1-first-run). Inside each, put the complete source code the LLM produced — even if most of it is duplicated from earlier stages.
  5. 05
    write run.yaml
    A single manifest at the root of your run directory. Only include stages you actually ran — no empty placeholders. Schema below.
  6. 06
    open a PR
    Fork, branch, push, open the PR against main. Get into the leaderboard as soon as it lands.
